# lqsblog

> 前后端分离

- 前端前台、前端后台、后端服务各自独立。

> 多语言版本

- 前端前台(Nuxt、Next、uni-app)、前端后台(Vue、React)、后端服务(JAVA、PHP、Node.js)。

> 技术栈应用

- 使用 Restful、JWT(自动刷新)、RBAC、微服务思想等技术开发。
